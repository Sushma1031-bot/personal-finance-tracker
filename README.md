# 💰 Personal Finance Tracker

A modern and responsive **Personal Finance Tracker** built to help users manage their daily expenses, monitor spending habits, and stay within budget. The application provides an intuitive dashboard with insightful visualizations, monthly summaries, and category-wise expense tracking.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Sushma1031-bot/personal-finance-tracker.git
```

Navigate into the project

```bash
cd personal-finance-tracker
```

Install dependencies

```bash
npm install
```

or

```bash
pnpm install
```

Run the development server

```bash
npm run dev
```

or

```bash
pnpm dev
```

Open your browser and visit:

```
http://localhost:5173
```
